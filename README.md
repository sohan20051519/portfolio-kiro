# Sohan A - Portfolio Website

A modern, minimalist portfolio website showcasing AI-driven development and DevOps expertise with neomorphic design and smooth animations.

## Overview

This is a personal portfolio website featuring a unique "vibe coding" aesthetic with custom animations, 3D tilt effects, and an ethereal fog canvas background. Built with vanilla JavaScript and Tailwind CSS for optimal performance.

## Features

- **Neomorphic Design**: Soft shadows and depth for a modern, tactile feel
- **Custom Cursor**: Desktop-only smooth cursor tracking with follower effect
- **3D Tilt Effects**: Interactive card tilting on hover
- **Fog Canvas Animation**: Ethereal particle system background
- **Text Scramble Effect**: Glitch-style text animation on hover
- **Floating Navigation**: Transforms into a pill-shaped bar on scroll
- **Scroll Reveal Animations**: Smooth fade-in effects for content sections
- **Fully Responsive**: Mobile-first design with hamburger menu
- **Performance Optimized**: Minimal dependencies, fast load times

## Tech Stack

- HTML5
- CSS3 (Tailwind CSS via CDN)
- Vanilla JavaScript
- Google Fonts (Cinzel & Manrope)

## Sections

1. **Hero** - Introduction with social links
2. **About** - Experience timeline and tech stack
3. **Projects** - Portfolio of selected works
4. **Certificates** - Professional credentials
5. **Contact** - Get in touch form
