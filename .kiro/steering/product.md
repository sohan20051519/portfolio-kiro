# Product Overview

Personal portfolio website for Sohan A, showcasing AI-driven development and DevOps expertise.

## Purpose

Professional portfolio targeting recruiters, clients, and developers to demonstrate:
- AI-augmented development capabilities ("Vibe Coding")
- Full-stack web development projects
- DevOps automation and cloud infrastructure skills

## Design Philosophy

**Neomorphic aesthetic** with ethereal, Halloween-themed elements:
- Soft shadows and depth for tactile feel
- Custom animations and 3D effects
- Minimalist, performance-focused approach
- Mobile-first responsive design

## Key Features

- Custom cursor with smooth tracking (desktop only)
- Fog canvas particle system background
- 3D tilt effects on interactive cards
- Text scramble animation on logo hover
- Floating navigation that transforms on scroll
- Scroll reveal animations with Intersection Observer
- Fully responsive with hamburger menu for mobile

## Target Audience

- Technical recruiters
- Potential clients for freelance work
- Fellow developers and collaborators
- Companies seeking AI/DevOps expertise
